# Ghostddit

Find posts and comments from any hidden Reddit profile — instantly.

## About Ghostddit — Reddit Hidden Posts & Comments Finder

**Ghostddit** is the fastest free tool to view hidden Reddit profiles, find hidden Reddit posts, and browse the full comment history of any Reddit user — even when their profile is completely hidden

## Repos

**[Ghostddit Extension](https://github.com/ghostddit/ghostddit-extension/):** Ghostddit Extension is a browser extension that reveals a Reddit user's posts & comments on their profile page.

## Open Source

Ghostddit is **open source**. We believe in transparency and community-driven development.
